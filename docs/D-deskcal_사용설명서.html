🚀 시작하기

프로그램 설치

  1. GitHub 릴리즈 페이지에서 최신 D-deskcal-v1.1.3-installer.zip 다운로드
  2. 압축 파일을 원하는 폴더에 압축 해제
  3. D-deskcal.exe 파일을 실행
  4. Windows Defender 또는 백신 프로그램에서 허용 설정

💡 팁

처음 실행 시 Windows에서 보안 경고가 나올 수 있습니다. "추가 정보" → "실행"을 클릭하여 실행하세요.

🎯 첫 실행

초기 설정

  1. 프로그램이 시작되면 바탕화면에 캘린더 위젯이 표시됩니다
  2. 시스템 트레이(우하단)에 D-deskcal 아이콘이 나타납니다
  3. 기본적으로 월별 보기로 현재 달이 표시됩니다

기본 조작

  • 날짜 이동: 상단 화살표 버튼 또는 마우스 휠
  • 오늘로 이동: "오늘" 버튼 클릭
  • 보기 전환: 우상단 보기 모드 버튼
  • 위젯 이동: 캘린더를 드래그하여 위치 조정

➕ 일정 추가

새 일정 만들기

  1. 원하는 날짜를 더블클릭 또는 우클릭 → "일정 추가"
  2. 일정 편집 창에서 정보 입력:
    • 제목: 일정 이름
    • 시간: 시작/종료 시간 (하루종일 옵션)
    • 설명: 자세한 내용
    • 캘린더: 저장할 캘린더 선택
  3. "저장" 버튼 클릭

💡 빠른 일정 추가

AI 기능을 활용하여 자연어로 일정을 추가할 수 있습니다.
예: "내일 오후 3시에 회의"

✏️ 일정 수정 및 삭제

기존 일정 편집

  1. 수정할 일정을 클릭하여 선택
  2. 일정을 더블클릭 또는 우클릭 → "수정"
  3. 일정 편집 창에서 내용 변경
  4. "저장" 버튼으로 변경사항 적용

일정 삭제

  • 개별 삭제: 일정 우클릭 → "삭제"
  • 키보드: 일정 선택 후 Delete 키

⚠️ 주의사항

삭제된 일정은 복구할 수 없습니다. 신중하게 선택하세요.

🔄 반복 일정

반복 일정 설정

  1. 일정 추가/수정 시 "반복" 옵션 선택
  2. 반복 유형 선택:
    • 매일: 매일 반복
    • 매주: 매주 같은 요일
    • 매월: 매월 같은 날
    • 매년: 매년 같은 날
    • 사용자 정의: 복잡한 반복 규칙
  3. 반복 종료 조건 설정 (날짜 또는 횟수)

💡 고급 기능

🔗 구글 캘린더 연동

구글 계정 연결

  1. 설정 창으로 이동 (시스템 트레이 우클릭 → "설정")
  2. "캘린더" 탭에서 "구글 캘린더 연동" 활성화
  3. "Google 인증" 버튼 클릭
  4. 웹 브라우저에서 Google 계정으로 로그인
  5. 권한 승인 후 자동으로 동기화 시작

동기화 기능

  • 양방향 동기화: D-deskcal ⇄ Google Calendar
  • 실시간 업데이트: 변경사항 즉시 반영
  • 다중 캘린더: 여러 구글 캘린더 지원
  • 오프라인 모드: 인터넷 없이도 로컬에서 작업

🛠️ 문제 해결

자주 묻는 질문

Q: 프로그램이 시작되지 않아요

A: Windows Defender 예외 목록에 추가하고, 관리자 권한으로 실행해보세요.

Q: 구글 캘린더 동기화가 안 돼요

A: 인터넷 연결 확인 후 Google 계정을 다시 인증해보세요.

Q: 위젯이 사라졌어요

A: 시스템 트레이 아이콘을 클릭하거나 프로그램을 재시작하세요.
