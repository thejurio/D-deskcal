설치 방법

1

인스톨러 다운로드

D-deskcal-v1.1.5-installer.zip 파일을 다운로드하고 압축을 해제합니다.

2

관리자 권한으로 실행

설치 파일을 마우스 우클릭 → "관리자 권한으로 실행"을 선택합니다.

3

Windows Defender 허용

첫 실행 시 Windows Defender 경고가 나타나면 "추가 정보" → "실행"을 클릭합니다.

주의: 설치 시 인터넷 연결이 필요합니다.

투명도 및 잠금 기능

1

투명도 조절

설정 → 외관 탭에서 투명도 슬라이더를 조절합니다.

2

잠금 모드

Ctrl 키를 눌러 잠금/해제를 전환합니다.

3

바탕화면 통합

투명도를 조절하여 바탕화면과 자연스럽게 어우러지도록 설정합니다.

팁: 잠금 모드에서는 창 이동이나 크기 조절이 불가능합니다.
7 / 15

일정 생성하기

1

기본 일정 생성

원하는 날짜를 더블클릭하면 일정 생성 창이 열립니다.

2

일정 정보 입력

제목, 시작/종료 시간, 설명을 입력합니다.

3

캘린더 선택

저장할 캘린더를 선택합니다 (Google 캘린더 또는 로컬 캘린더).

4

저장

"저장" 버튼을 클릭하여 일정을 생성합니다.

8 / 15

AI 일정 생성 (고급 기능)

1

API 키 설정

설정 → 계정 탭에서 Gemini API 키를 입력합니다.

2

AI 일정 추가

Ctrl + K를 누르거나 우클릭 메뉴에서 "AI로 일정 추가"를 선택합니다.

3

자연어 입력

"내일 오후 2시에 병원 예약"과 같이 자연스럽게 입력합니다.

AI 입력 예시:

  • "매주 월요일 오전 9시에 회의"
  • "다음주 금요일 저녁 7시에 친구와 저녁식사"
  • "12월 25일 크리스마스 파티"
9 / 15

반복 일정 설정

1

반복 옵션 선택

일정 생성/수정 창에서 "반복" 체크박스를 선택합니다.

2

반복 패턴 설정

매일, 매주, 매월, 매년 중 원하는 패턴을 선택합니다.

3

종료 조건 설정

특정 날짜까지 또는 특정 횟수만큼 반복하도록 설정합니다.

주의: 반복 일정을 삭제할 때는 "이 일정만", "이후 모든 일정", "모든 반복 일정" 중 선택해야 합니다.

문제 해결

?

Google 로그인이 안 될 때

설정 → 계정 → "다시 로그인" 버튼을 클릭하여 재인증합니다.

?

동기화가 안 될 때

인터넷 연결을 확인하고, 설정에서 동기화 간격을 조정해보세요.

?

프로그램이 멈출 때

작업 관리자에서 프로그램을 종료하고 재시작합니다.

?

투명도가 적용되지 않을 때

Windows 설정에서 "투명 효과" 옵션이 켜져 있는지 확인합니다.

유용한 팁

💡 효율적 사용법

  • 자주 사용하는 일정은 반복 설정 활용
  • 다중 모니터에서는 보조 모니터에 배치
  • 바탕화면 색상과 조화로운 투명도 설정

⚡ 단축키 활용

  • Ctrl: 잠금/해제
  • Ctrl + K: AI 일정 추가
  • F5: 수동 동기화

🔄 백업 및 복원

  • 설정은 settings.json 파일에 저장
  • 로컬 일정은 calendar.db에 저장
  • 정기적인 백업 권장
